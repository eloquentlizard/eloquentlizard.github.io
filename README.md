# eloquentlizard's personal tiddlywiki notebook project
This repository is simply a way for me to work on and publish a tiddlywiki so I may share it with whomever I so choose. It's far from perfect, and will take some time to get back to the state that older tiddlywiki projects I've made in the past have achieved, but I'm in no rush and I think starting fresh with no additional baggage or older version bloat means I'll have more freedom to make this something better for me than other social media projects. 

## questions one might have
- **Is this the most elegant or secure solution?** No.
- **Do you care about that?** Also no.
- **What is the goal here?** To jot things down, take notes, wrote homebrew, brainstorm projects, and be able to share any of it easily.
